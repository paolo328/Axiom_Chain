Pectra Staking Manager:
Ethereum Validator Management Tool

A consumer-facing web application enabling Ethereum validators to access new Pectra upgrade features — built as a joint initiative between Axiom Chain, PierTwo, and Hashlock, with validator consolidation as the flagship capability.

Stack

React, Next.js, Ethereum

Role

Design & Frontend Development

Scope

Design, Frontend, QA

The Mission

“To give institutional stakers a dedicated platform for managing validators — including access to new Ethereum protocol capabilities as they land at the consensus layer.”

Core Objectives

  • ✓Validator consolidation via EIP-7251 mechanics
  • ✓EIP-7702 transaction batching for multi-step operations
  • ✓Clean UX for non-technical stakers managing complex validator lifecycles
  • ✓Joint delivery across Axiom Chain, PierTwo, and Hashlock

The Challenge

Ethereum's Pectra upgrade introduced significant new validator management capabilities — most notably the ability to consolidate multiple validators into a single one under EIP-7251 — but no user-facing tool existed to access these features at launch. Institutional stakers needed a usable interface from day one without resorting to raw contract interactions.

EIP-7702 transaction batching support was required to streamline multi-step validator operations, and the app had to work across wallet types and varying validator configurations. QA cycles were tied directly to PierTwo's live validator infrastructure, requiring close coordination across two companies' tooling and Notion environments.

What We Built

Validator Consolidation Flow

The flagship feature — a guided flow allowing validators to consolidate multiple validators into one using EIP-7251 mechanics, abstracting the underlying contract complexity into a clean step-by-step interface.

EIP-7702 Transaction Batching

Multi-step validator operations batched into single transactions, reducing friction and gas overhead for institutional users performing bulk validator management actions.

Wallet Connection & Validator Overview

Primary entry points for the app — wallet connection supporting multiple wallet types, and a validator viewing experience showing current configuration and available Pectra actions.

QA & Collaborative Delivery

End-to-end QA process coordinated directly with PierTwo's validator infrastructure team, with a shared review cycle across Axiom Chain, PierTwo, and Hashlock ensuring production readiness.

Technical Architecture

Consumer web app built in Next.js and React, deployed via Vercel, targeting Ethereum mainnet and testnets post-Pectra. Integrates EIP-7251 consolidation contracts and EIP-7702 batching logic via standard wallet providers. Product design owned by Axiom Chain in Figma; smart contract and backend infrastructure owned by PierTwo; security review by Hashlock. Coordinated across shared Notion, Slack, and Telegram.
