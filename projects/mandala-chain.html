Mandala Chain:
DeFi Infrastructure Suite — DEX, Staking & Lending

A comprehensive DeFi infrastructure buildout spanning a Uniswap V2 DEX, staking and vesting systems, an Aave V3 lending fork, and hub integration work across the Mandala Chain ecosystem.

Stack

Next.js, Solidity, Hardhat, Wagmi

Role

Full DeFi Stack Delivery

Scope

DEX, Lending, Staking, Vesting, Hub Integration

The Mission

“To establish a complete, production-ready DeFi stack on Mandala Chain that supports trading, lending, staking and token distribution.”

Core Objectives

  • ✓Deploy a Uniswap V2 DEX with modern frontend for swap and liquidity management
  • ✓Launch an Aave V3 lending market with conservative risk parameters and admin tooling
  • ✓Build staking and vesting platforms integrated into the Mandala Hub
  • ✓Coordinate multi-workstream delivery across independent protocol engagements

The Challenge

Mandala Chain needed a full suite of DeFi primitives to attract users and liquidity. Each component — DEX, lending, staking, vesting — required careful adaptation to the target chain. The ecosystem hub needed to integrate multiple independently built products into a coherent user experience. Protocol safety, liquidation management and operational oversight were essential for a lending market launch.

Forking Uniswap V2 required both contract deployment and a decision between legacy frontend or a modern rebuild. The Aave V3 fork demanded conservative risk parameters, oracle integration and phased delivery to manage protocol health. Vesting and staking flows handled significant funds, requiring careful security consideration during hub integration. Multiple concurrent workstreams had to be coordinated across separate proposals and delivery phases.

What We Built

Uniswap V2 DEX

Core and periphery contracts deployed via Hardhat — paired with a modern Next.js frontend covering swap, pool creation and liquidity management.

Aave V3 Lending Protocol

Full Aave V3 fork configured with lending markets, risk parameters and oracle integration — plus an admin dashboard for manual liquidation, pause controls and emergency operations.

Staking & Vesting Platforms

Token vesting claim wizard and staking pools dashboard built with shared Wagmi and Polkadot wallet infrastructure, integrated directly into the Mandala Hub.

Administration & Liquidation Tooling

Operational dashboard with manual liquidation controls, pause/unpause functionality and automated liquidation monitoring — purpose-built for lending market safety.

Technical Architecture

Uniswap V2 core and periphery contracts deployed with Hardhat alongside a modern Next.js swap frontend. Aave V3 fork with configured lending markets, risk parameters and oracle integration. Admin dashboard with manual liquidation, pause/unpause and emergency controls. Token vesting claim wizard and staking pools dashboard integrated into Mandala Hub via shared Wagmi wallet config, Polkadot wallet support and session auth.
